# passgen

Simple CLI utility to generate complex passwords. Written in python3.

## Installation

```sh
git clone https://github.com/Archetypum/passgen.git
cd passgen/
sudo make  # or doas make
```

## Usage

```sh
man passgen
# or
passgen --help
```
